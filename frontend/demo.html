Integrate

Three lines, not three services.

No audio pipeline to stand up. The SDK handles the microphone, the request, and playback.

const sdk = new VoiceSDK({ apiKey, baseUrl }); await sdk.startListening('am'); // mic turns on const { text } = await sdk.stopListening(); // mic turns off, transcript returns await sdk.speak(text, 'am'); // spoken back to the user
Languages
